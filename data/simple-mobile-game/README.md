# Tap Blast

간단하고 중독성 있는 모바일 웹 게임입니다.

## 플레이 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. **게임 시작** 버튼을 누릅니다.
3. 떨어지는 원형 타겟을 빠르게 터치해서 점수를 올립니다.
4. 목표는 목숨이 모두 사라지기 전에 최대한 많은 점수를 얻는 것입니다.

## 주요 특징

- 터치 중심 모바일 UI
- 효과음 지원 (히트, 실패, 버튼)
- 3단계 난이도 선택: 쉬움 / 보통 / 어려움
- 최고 점수 랭킹 저장 및 표시
- 콤보 보너스 시스템
- 시간이 지날수록 빨라지는 난이도
- 데스크톱 및 모바일 브라우저에서 실행 가능
