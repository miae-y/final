# GUESS 코리아 - 온라인 수익성 개선 및 브랜드 재구축전략 포털

본 프로젝트는 게스 코리아(GUESS Korea)의 홈쇼핑/아울렛 할인 중심 구조를 타파하고 브랜드 단가(ASP) 정상화 및 MZ 세대 유입을 달성하기 위한 **온라인 수익성 개선 및 브랜드 턴어라운드 전략 마스터플랜**을 시각화한 양방향 포털 대시보드 웹 애플리케이션입니다.

---

## 📂 프로젝트 폴더 구조 (Project Structure)

```text
my-project/
├── index.html          # 메인 엔트리 포인트 (로그인 화면 및 대시보드 뷰 홀더)
├── package.json        # 프로젝트 메타데이터 정의
├── README.md           # 프로젝트 문서화 가이드
└── src/
    ├── app.js          # 로그인 로직 및 탭 전환 상호작용 스크립트
    ├── styles.css      # 브랜드 아이덴티티가 가미된 스타일시트
    └── assets/
        └── guess_login_bg.png  # 고해상도 브랜드 이미지 (배경용)
```

## ✨ 주요 기능 및 특징 (Key Features)

1. **프리미엄 로그인 스크린 (Login Screen)**
   - 게스의 역삼각형 심볼과 물음표(?) 로고를 SVG 벡터 그래픽으로 고감도 시각화.
   - 고품질 흑백 패션 이미지 배경과 글래스모피즘(Glassmorphism) 스타일의 흐림(blur) 효과 카드 설계.
   - 계정 및 비밀번호 검증 오류 시 미세 흔들림(shake) 애니메이션 피드백 제공.
   - 기본 테스트 관리자 정보 제공으로 누구나 쉽게 접근 가능.

2. **양방향 대시보드 포털 (Interactive Admin Dashboard)**
   - 로그인 성공 시 부드러운 페이드인(Fade-in) 뷰 전환.
   - 사이드바 내비게이션 메뉴를 통한 실시간 동적 콘텐츠 영역 교체.
   - **전략 마스터플랜 탭:** 현 구조의 한계점 진단 카드 및 핵심 매출/턴어라운드 비교 분석표 제공.
   - **유통 채널 3원화 탭:** 플랫폼 간 카니발라이제이션 차단을 위한 무신사/29CM, 백화점 종합몰, 오픈마켓 3단 카드 레이아웃.
   - **상품 혁신 & 로드맵 탭:** 레트로 아카이브 협업, 카테고리 쇄신 상세 카드 및 3단계 런칭 가이드라인 타임라인.

3. **반응형 웹 디자인 (Responsive Design)**
   - 데스크톱 화면에서 최적화된 사이드바 그리드 레이아웃을 제공하며, 태블릿 및 모바일 환경에서도 가독성이 무너지지 않는 세로형 스택 그리드를 적용하였습니다.

웹 포털에 로그인하기 위한 테스트 관리자 계정 정보는 다음과 같습니다:

- **비밀번호(PW):** `guess`

## 🚀 실행 방법 (How to Run)

본 애플리케이션은 별도의 로컬 서버 설치나 빌드 과정 없이 **웹 브라우저에서 즉시 실행**이 가능하도록 순수 HTML, CSS, JavaScript로 구현되어 있습니다.

1. `my-project/` 폴더 내의 `index.html` 파일을 더블 클릭하여 기본 웹 브라우저(Chrome, Edge, Safari 등)로 엽니다.
2. 안내된 관리자 계정(`admin` / `guess`)을 입력하여 포털 내부 대시보드로 원활하게 진입해 보세요.
